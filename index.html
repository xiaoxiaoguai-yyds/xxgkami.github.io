使用方法

步骤1: 下载并部署

从GitHub克隆项目并部署到支持PHP和MySQL的服务器环境

git clone https://github.com/xxg11/xxg-kami-push.git

步骤2: 配置数据库

导入install.sql创建数据库结构，修改config.php配置数据库连接信息

步骤3: 生成卡密

登录管理后台，批量生成卡密，可选择时间卡密或次数卡密

步骤4: API调用示例

在您的应用中集成卡密验证API

// PHP示例代码
$url = "https://您的域名/api/verify.php";
$data = [
    'card' => '用户输入的卡密',
    'device_id' => '设备唯一标识'
];

$options = [
    'http' => [
        'header' => "Content-type: application/x-www-form-urlencoded\r\n",
        'method' => 'POST',
        'content' => http_build_query($data)
    ]
];

$context = stream_context_create($options);
$result = file_get_contents($url, false, $context);
$response = json_decode($result, true);

if ($response['code'] == 0) {
    // 卡密验证成功
    echo "验证成功，到期时间: " . $response['data']['expire_time'];
} else {
    // 验证失败
    echo "验证失败: " . $response['msg'];
}
